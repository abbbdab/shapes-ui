# Toggle

A button that can be toggled between two states - on or off. Useful for features like bold/italic text formatting or enabling/disabling settings.

## Features

- **Single Toggle**: A single button that toggles between pressed and unpressed states
- **Toggle Group**: Multiple toggles grouped together with shared styling
- **Size Variants**: Small, default, and large sizing options
- **Style Variants**: Default and outline styles
- **Icon Support**: Built-in support for icons with automatic sizing

## Props

### Toggle

The `Toggle` component accepts all standard button props plus the following:

- `variant`: `"default" | "outline"` - The visual style of the toggle
- `size`: `"sm" | "default" | "lg"` - The size of the toggle
- `defaultPressed`: `boolean` - Whether the toggle is initially pressed
- `onPressedChange`: `(pressed: boolean) => void` - Callback when pressed state changes
- `children`: React elements to display inside the toggle

### ToggleGroup

The `ToggleGroup` component accepts all group props plus:

- `children`: Multiple `Toggle` components

## Examples

### Basic Toggle

```tsx
import { Toggle } from "@/components/ui/toggle";
import { Bold } from "lucide-react";

export function BasicToggle() {
  return (
    <Toggle>
      <Bold />
      Bold
    </Toggle>
  );
}
```

### Toggle Group

```tsx
import { Toggle, ToggleGroup } from "@/components/ui/toggle";
import { Bold, Italic, Underline } from "lucide-react";

export function TextFormattingToolbar() {
  return (
    <ToggleGroup>
      <Toggle>
        <Bold />
      </Toggle>
      <Toggle>
        <Italic />
      </Toggle>
      <Toggle>
        <Underline />
      </Toggle>
    </ToggleGroup>
  );
}
```

### With Size Variants

```tsx
import { Toggle } from "@/components/ui/toggle";

export function SizedToggles() {
  return (
    <>
      <Toggle size="sm">Small</Toggle>
      <Toggle size="default">Default</Toggle>
      <Toggle size="lg">Large</Toggle>
    </>
  );
}
```

### Controlled Toggle

```tsx
"use client";

import { useState } from "react";
import { Toggle } from "@/components/ui/toggle";
import { Bold } from "lucide-react";

export function ControlledToggle() {
  const [isBold, setIsBold] = useState(false);

  return (
    <Toggle
      pressed={isBold}
      onPressedChange={setIsBold}
    >
      <Bold />
      {isBold ? "Bold (on)" : "Bold (off)"}
    </Toggle>
  );
}
```

## Styling

The toggle component uses Tailwind CSS utilities and CVA for styling. You can customize the appearance by:

1. Using the `variant` and `size` props
2. Passing custom `className` props
3. Overriding styles in your CSS

## Accessibility

- Proper ARIA labels for screen readers
- Keyboard navigation support (Enter/Space to toggle)
- Focus states for keyboard users
- Semantic HTML button elements
